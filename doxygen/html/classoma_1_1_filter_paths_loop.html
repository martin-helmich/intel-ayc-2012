Detailed Description

Loop body for filtering travels by minimal costs.

This loop body filters a set of travels by a predefined minimal cost. It takes an input vector "in" and a "CostRange" object pointer as arguments and fills its output vector "out" with all travels from "in" that are possibly cheaper than defined by the cost range.
